# WikiRush (Arbeitstitel)
Ein Multiplayer-Browsergame, in dem du durch Wikipedia-Artikel navigierst, um das Ziel schneller als deine Gegner zu erreichen.

## 📌 Features
- 🎮 **Singleplayer-Modus**: Spiele alleine gegen die Zeit
- 🌍 **Multiplayer mit ELO-Ranking**: Spiele online gegen andere & steige im Ranking auf
- 🎉 **Party-Modus**: Lade Freunde ein und spiele privat
- 🔗 **Wikipedia API**: Zufällige Start- & Ziel-Artikel
